# helloworld
web访问输出 hello world, 并可获取到客户端ip地址 , 服务器ip地址和主机名；适合用django做实验,例如用作docker、kubernetes的简单demo使用

## 环境说明
#### os: CentOS 7
#### Python3: Python36
#### Django: 2.0

### 示例
#### [root@master ~]# curl http://10.0.6.2
#### hello world! Your ip is: 10.0.6.2, ServerHostname: DESKTOP-OCO3THD, ServerIP: 10.0.6.2
